This program implements Scoreboards.
A input file named "input.txt" will consist under 10 instructions.
The output which is a scoreboard for every round, will be shown on the user's screen.

Sample input file:
L.D F6, 34(R2)
L.D F2, 45(R3)
MUL.D F0, F2, F4
SUB.D F8, F6, F2
DIV.D F10, F0, F6
ADD.D F6, F8, F2
